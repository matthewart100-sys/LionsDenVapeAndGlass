Option 1: Simple Auto-Reload (Recommended)

The easiest way - no installation needed!
Steps:
  • Open your website in the browser
  • Press F12 to open DevTools
  • Go to the Console tab
  • Copy this script and paste it:
    (function() { console.log('üîÑ Auto-Reload enabled!'); let lastContent = document.documentElement.innerHTML; setInterval(() => { fetch(window.location.href, {cache: 'no-store'}) .then(r => r.text()) .then(text => { if (lastContent !== text) { console.log('‚úÖ Changes detected! Reloading...'); lastContent = text; location.reload(); } }) .catch(e => console.log('Check:', e.message)); }, 1000); })();
  • Press Enter
  • ‚úÖ Done! The page will auto-reload whenever you save changes
üí° Tip: Keep your DevTools open (F12) and refresh code as you make changes. The auto-reload will trigger whenever your files change!

Option 2: Dev Server (If you want more control)

Use your local Flask development server:
Steps:
  • Open PowerShell/Terminal
  • Navigate to your project folder
  • Run this command:
    python -m http.server 8000
  • Visit: http://localhost:8000
  • ‚úÖ Your changes will reflect automatically!
üí° Note: If you're using Flask backend, use port 5000 instead:
cd backend && python app.py

Option 3: VS Code Live Server Settings

Configure Live Server for auto-reload:
Steps:
  • Open VS Code Settings (Ctrl + ,)
  • Search for: liveServer.settings.root
  • Set it to your project root
  • Search for: liveServer.settings.donotShowInfoMsg
  • Enable it (check the box)
  • Right-click your HTML file and select Open with Live Server
  • ‚úÖ Live Server will auto-reload on save!
